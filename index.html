Sobre o Felipe

Sou Felipe Macedo Ribeiro, um profissional em transição de carreira para a área de tecnologia. Atualmente, estou me formando em Análise e Desenvolvimento de Sistemas e participo de bootcamps intensivos em Fullstack Java e Desenvolvimento Web. Minha jornada inclui o desenvolvimento de projetos práticos, como sistemas de e-commerce, APIs de RH e aplicações de delivery, utilizando tecnologias como Java, Spring Boot, SQL e JavaScript, além de habilidades em Frontend com HTML, CSS e frameworks modernos.

O autodesenvolvimento sempre foi uma prioridade para mim. Busco constantemente aprimorar minhas habilidades através de cursos, desafios práticos e metodologias ágeis. Estou focado em evoluir na programação orientada a objetos, desenvolvimento de APIs, modelagem de banco de dados e criação de interfaces responsivas e interativas. Minha paixão pela tecnologia me motiva a enfrentar novos desafios e a contribuir com soluções inovadoras para o mercado.
